ğŸ§© Autodesk Fusion 360 â€“ Robotic Arm Design

File Name: Food_Handling_Roboticarm_Design

Software Used: Autodesk Fusion 360

The robotic arm used in this project was designed and modeled using Autodesk Fusion 360. The software was selected for its powerful parametric modeling tools, easy assembly management, and accurate visualization capabilities, which are well suited for robotic system design.

The complete robotic arm design includes:

Base structure

Shoulder, elbow, and wrist joints

End-effector (mechanical gripper)

Servo motor mounting brackets

Linkages and joint connections

Each component was modeled individually and then assembled to ensure proper alignment, range of motion, and mechanical stability. Joint constraints and motion limits were applied to simulate real-world arm movements and to verify reachability and workspace.

Roboslice is a web application developed to manage food orders digitally and integrate seamlessly with the automated food handling system.
